Z_ALV_REQUEST - Проверка порядка запросов перед транспортом
-----------------------------------------------

Домен - это надстройка над стандартным типом САП. Является необязательным.
Может ограничивать допустимые значения через диапазон(ы) (интервалы значений), через конкретное указание(перечисление) допустимых значений. На основе домена можно создать элемент данных.

Элемент данных - является обязательным уровнем типа. Позволяет использовать домен, привязывать средство поиска, прописать текст отображения.

Кластеры ракурсов - предназначены для объединения нескольких ракурсов. 
Для создания и изменения кластеров ракурсов используется транзакция se54.
